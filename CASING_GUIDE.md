# Casing Guide

The following is a guide of the different casing styles users can choose. 
File casing helps maintain a consistent style across a file hierarchy. 


## snake_case (default)
Snake case separates each word with an underscore character (_). All letters need to be lowercase (a screaming snake case is where all letters are capital). This will be the default
<br>
Some examples include
* in_class_demo.py
* ai_presentation_ethics.pptx
* naca_student_presentation_guidelines.pdf

When choosing this option in our script, type "snake" as your case.

## kebab-case
Kebab-case is very similar to sname_case, except that it separates each word with a dash character (-) instead of an underscore.
So, all words are lowercase, and each word gets separated by a dash
<br>
Some examples include
* in-class-demo.py
* ai-presentation-ethics.pptx
* naca-student-presentation-guidelines.pdf

When choosing this option in our script, type "kebab" as your case.

## camelCase
camelCase starts by making the first word lowercase. Then, it capitalizes the first letter of each word that follows.
<br>
Some examples include
* inClassDemo.py
* aiPresentationEthics.pptx
* nacaStudentPresentationGuidelines.pdf

When choosing this option in our script, type "camel" as your case.

## PascalCase
PascalCase is very similar to camelCase, except the first letter of the first word to also be capitalized.
<br>
Some examples include
* InClassDemo.py
* AiPresentationEthics.pptx
* NacaStudentPresentationGuidelines.pdf

When choosing this option in our script, type "pascal" as your case.

## lowercase with spaces
lowercase with spaces is much like snake_case, except it separates words with spaces ( ) instead of an underscore. 
So, all words are lowercase, and each word gets separated by a space
<br>
Some examples include
* in class demo.py
* ai presentation ethics.pptx
* naca student presentation guidelines.pdf

When choosing this option in our script, type "lower" as your case.

## Title Case With Spaces
Title Case with is much like lowercase with spaces, except it capitalizes the first letter in every word. 
<br>
Some examples include
* In Class Demo.py
* Ai Presentation Ethics.pptx
* Naca Student Presentation Guidelines.pdf

When choosing this option in our script, type "title" as your case.
